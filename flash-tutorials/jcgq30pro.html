一、刷机教程 ​

JCG Q30 Pro 的刷机过程非常简单，不需要拆机，直接刷入237大佬的过渡固件就可以完成刷机。

浏览器输入192.168.10.1

进入路由器升级页面，选择刷机包中的过渡固件升级，然后“升级固件”，取消勾选保留配置，

将下载好的过渡固件 immortalwrt_mediatek_mt7981_mt7981_spim_nand_rfb_squashfs_sysupgrade 上传，

等到检测完成后，直接点击“升级”，等待几分钟等路由器重新开机，就完成了过渡固件的刷入！

  • 过渡固件的后台为 192.168.6.1，账号 root 密码 password

  • 不太讲究的话，可以直接使用过渡固件，如果有更多的需求，那就要通过 U-Boot 刷入其他的固件，推荐使用别的固件包。

这个固件没有带插件，不能做到开箱即用，刷个带插件的固件吧!

  • 过渡固件开机后，会在后台自动刷入 U-Boot，进入 U-Boot，需要手动指定静态 IP ，才能进入 U-Boot！

  • 进入uboot

断电（拔掉路由器通电源）后，按住路由器的：Reset按键，再接入电（插上通电源）等10秒后，灯变蓝，就进入了U-boot。 手动将电脑ip设置为 192.168.1.0/24 网段的ip，例如 192.168.1.2，就可以进入后台：192.168.1.1

二、刷入新版U-Boot ​

使用到的工具：SSH连接工具

这里我就以MobaXterm工具为例，打开SSH工具，填入路由器管理后台地址----设置登录用户为root----点击 OK 确定即可。

通过 ssh 连接路由器，过渡固件的默认用户名和密码为 root 和 password。

把uboot固件上传到路由器的 /tmp 目录下：

输入以下命令：

cd /tmp

mtd write /tmp/mt7981-jcg_q30-fip-expand.bin FIP

刷入新U-Boot后你就可以刷入其他各种固件了。
