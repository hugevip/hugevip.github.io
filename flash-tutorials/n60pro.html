一、解锁 SSH

磊科N60PRO原厂自带SSH，直接进入系统后台：192.168.0.1，在路由器里开启ssh。

使用SSH工具（推荐MobaXter）连接路由器（192.168.0.1），用户名：useradmin 密码：管理密码，登录成功代表ssh开启！

二、2、备份原厂系统文件

#查看分区 cat /proc/mtd

#备份 BL2 引导分区 dd if=/dev/mtd1 of=/tmp/mtd1_BL2.bin

#备份 u-boot-env 环境变量 dd if=/dev/mtd2 of=/tmp/mtd2_ubootenv.bin

#备份 Factory 校准分区 dd if=/dev/mtd3 of=/tmp/mtd3_Factory.bin

dd if=/dev/mtd4 of=/tmp/mtd4_fip.bin

#备份 ubi 系统固件，这个时间会长一点 dd if=/dev/mtd5 of=/tmp/mtd5_ubi.bin

备份完了之后，进入/tmp目录，下载这四个文件进行备份。

三、刷入Uboot

使用SSH工具上传要刷入的UBOOT文件到 tmp 文件夹，上传成功使用以下命令刷入Uboot，

cd /tmp

mtd write /tmp/mt7986-netcore_n60-pro-fip.bin FIP

  • 刷入完成后断电，按住 reset 键通电，等待 8s 后电源指示灯重新变成蓝色，直接访问 192.168.1.1，出现U-Boot界面说明刷入成功，这款U-Boot 自带DHCP，不需要手动修改电脑 IP，特别适合新手小白！

  • 选择 immortalwrt固件上传，点击 Update 耐心等待刷入即可。

  • 固件刷入完成后，访问 192.168.1.1，能成功进入后台就大功告成了。

237大佬的高功率固件也挺好用，唯一的缺點是128MB閃存太小了，刷好固件，軟體包可用空間只有60MB左右，安裝不了幾個外掛程式。

分享一下如何把外掛程式安裝到U盤裡，並且如果你的固件沒有istore商店，也可以安裝一下，方便使用。

  • 安裝istore商店 SSH连接 路由器后，逐条输入下面代码：

opkg update || exit 1 cd /tmp wget https://github.com/linkease/openwrt-app-actions/raw/main/applications/luci-app-systools/root/usr/share/systools/istore-reinstall.run chmod 755 istore-reinstall.run ./istore-reinstall.run

安装完成后退出登录、重新进入后台，就有istore商店了。

  • 最后如果感觉512MB内存不够使用，又不想购买2G内存版本，可以开启Swap，Linux 中的交换分区 / 交换空间，相当于 Windows 的 “虚拟内存”，是硬盘划分的一块区域，当物理内存（RAM）不足时，系统会把内存中暂时不用的数据移到 Swap 中，释放物理内存给活跃程序使用。

  • 使用外置U盘扩容overlay分区，带有USB接口的OP路由器都可以用这个方法。

使用DiskGenius工具U盘格式化为EXT4格式，分区表类型要GUID格式(即GPT)，建议把U盘分成3个分区 ： 一个5G左右给软件包/overlay使用， 一个1G给虚拟内存使用，最后一个分区给网络共享（轻NAS）使用。

插入U盘，来到系统→挂载点→已挂载的文件系统，点击卸载分区，先卸载挂载好的U盘。 挂载点→添加一个挂载点，UUID选择自己的U盘，挂载点“作为外部 /overlay 使用” 保存并应用最后，重启路由器，即可生效。

1.创建 swap 文件或使用分区（假设你使用分区 /dev/sda2）： mkswap /dev/sda2

2.启用 swap 分区： swapon /dev/sda2

3.查看 swap 状态： swapon -s free -h

4、开机自动挂载（编辑 /etc/fstab 添加一行） /dev/sda2 none swap sw 0 0
