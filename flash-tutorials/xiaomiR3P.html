一、开启SSH ​

  • 1、第一步登录后台192.168.31.1降级到开发版固件，然后打开小米ssh开启工具，点击一键开启ssh.bat按照视频步骤开启就可以了!

       能备份固件就备份，不能就到ssh里面备份！

  • 2、备份路由器固件

       查看分区情况

   cat /proc/mtd

   插个u盘到路由器上，查看以下

   df -h

  我的显示u盘如下：

  /dev/sda2      29.3G   4.3M   29.3G   0% /extdisks/sda2

备份全部分区数据（sda2是我的u盘根据你的修改）

dd if=/dev/mtd0 of=/extdisks/sda2/ALL.bin

dd if=/dev/mtd1 of=/extdisks/sda2/Bootloader.bin

dd if=/dev/mtd2 of=/extdisks/sda2/Config.bin

dd if=/dev/mtd3 of=/extdisks/sda2/Bdata.bin

dd if=/dev/mtd4 of=/extdisks/sda2/Factory.bin

dd if=/dev/mtd5 of=/extdisks/sda2/crash.bin

dd if=/dev/mtd6 of=/extdisks/sda2/crash_syslog.bin

dd if=/dev/mtd7 of=/extdisks/sda2/reserved0.bin

dd if=/dev/mtd8 of=/extdisks/sda2/kernel0.bin

dd if=/dev/mtd9 of=/extdisks/sda2/kernel1.bin

dd if=/dev/mtd10 of=/extdisks/sda2/rootfs0.bin

dd if=/dev/mtd11 of=/extdisks/sda2/rootfs1.bin

dd if=/dev/mtd12 of=/extdisks/sda2/overlay.bin

二、刷入pb-boot或者breed ​

1、上传pb-boot到路由器的tmp目录输入下面的命令开刷：

cd /tmp

mtd -r write /tmp/pb-boot-r3p.img Bootloader

刷入后，机器会重新启动，重启完后路由器亮黄灯，到此pb-boot刷入成功。

然后拔掉电源，用硬物顶住路由的reset键然后在插入电源开机，

等到路由的黄灯闪烁的时候，松开reset键，

在电脑上浏览器中输入192.168.1.1，就进入pb-boot控制台了。

2、刷入breed

mtd -r write /tmp/breed-mt7621-xiaomi-r3g-r3p.bin Bootloader

breed我没刷恩山说镁光芯片容易变砖，还有说NAND FLASH有坏块也会变砖，刷机有风险，变砖自己承担！

三、Padavan固件和OpenWrt固件 ​

1、刷Padavan

在电脑上浏览器中输入192.168.1.1，就进入pb-boot控制台，选择下载的Padavan固件包，点击恢复固件，开始刷机。

padavan

192.168.2.1

管理界面账户/密码：admin/admin

2、刷入OpenWrt

一定不要直接刷入openwert固件，可能变砖，先x-wrt过渡包在升级openwrt

在pb-boot中先刷入一个x-wrt固件过度，这是临时固件，无法保存任何设置，不能使用，记住了，刷完x-wrt后，浏览器登陆后台

192.168.15.1，密码：admin

在web界面中“系统-备份/升级中选择open-wrt (sysupgrade)固件升级，不保留配置！”即可平稳过度到openwrt固件

登陆web界面后记得接上wan口网线。

x-wrt

固件无线默认名称：X-WRT_XXXX，密码：88888888

固件管理界面：http://192.168.15.1/

管理界面账户/密码：admin/admin

SSH登录账户/密码：root/admin

kwrt

后台: 10.0.0.1 或 kwrt/ 密码: root

四、刷回原厂 ​

1、下载R3P开发版及ssh开启包，准备putty ssh工具 2、路由器断电，按住reset加电3秒黄灯闪所，浏览器192.168.1.1进入pb-boot界面，刷入开发版固件 3、开发版固件改名为miwifi.bin并复制到U盘，路由断电，按住reset不松开，路由加电等待刷入开发版 4、删除U盘开发版，并把ssh开启包和uboot（如果是用刷机工具备份的，在刷机工具下的data文件夹，mtd1.bin）复制到U盘，同3方式开启ssh 5、路由开发版固件启动，通过wifi连接并设置路由密码 6、ssh（192.168.31.1）进入路由，使用如下命令刷入uboot，即可完成回厂 mtd write /extdisks/sda1/mtd1.bin Bootloader
