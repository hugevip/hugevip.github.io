百元小米AX3000T路由器，开箱刷机详细教程 ​

  • 路由器版本：AX3000T RD03
  • 固件版本： 1.0.84

一、开启SSH功能

说明：AX3000T靠近电源口为WAN口，其他口为LAN口

网线一端插入LAN口，一端插入电脑。

运行run.bat ，输入2回车，输入路由器管理后台密码(输入过程不显示密码)回车，显示SSH server are activated，开启成功。

备份固件

运行脚本后，输入4回车。

会自动将原版固件备份到工具下的backups文件夹中。

二、刷入Uboot

使用到的工具：SSH连接工具

这里我就以MobaXterm工具为例，打开SSH工具，填入路由器管理后台地址----设置登录用户为root----点击 OK 确定即可。

小米路由器SSH用户名：root

小米路由器SSHt密码：root

写入Uboot

将下载的Uboot直接拖拽传输至tmp文件夹。

执行以下命令：

cd ../tmp #进入tmp文件夹

mtd write mt7981_ax3000t_an8855-fip-fixed-parts.bin FIP #写入Uboot文件

三、刷入OPENWRT固件

设置电脑IP为 192.168.1.2 掩码为 255.255.255.0

路由器进入Uboot

用卡针顶住路由器reset键，之后路由器接入电源。reset键一直按住等待10秒左右，指示灯变为蓝色后松开reset键。

浏览器登录192.168.1.1 ，即可进入Uboot页面，选择qwrt模式，

上传openwrt固件即可。
