一、恢复出厂设置 ​

  • 小米路由器默认IP地址：192.168.31.1，路由器先恢复一下出厂设置，设置好密码12345678，记住密码等会要用到。

  • 我的路由器版本是2.26.39稳定版，我的教程不需要刷入开发版，直接开刷，省时省力！

二、刷机教程 ​

1， 开启ssh，用专用工具开启，跟着视频操作就可以。

2，putty登录路由器

  • 找一个u盘导入固件，插入U盘

  • 进入U盘根目录运行命令：cd /extdisks/sda1

  • 备份固件，查看分区命令：cat /proc/mtd，执行备份命令（0-12），备份至U盘/extdisks/sda1目录。 没有u盘的可以备份至tmp目录，使用winscp下载，协议选择scp。

3，#刷入pb-boot，先执行以下命令

  • nvram set uart_en=1
  • nvram set flag_last_success=1
  • nvram set boot_wait=on
  • nvram commit
  • 前面4行代码是用来开启串口，如果不幸变砖，还可以通过串口来恢复。
  • 然后刷入pb-boot
  • cd /tmp
  • mtd write pb-boot.img Bootloader
  • reboot
  • 然后断电路由器，按住重置按钮并打开路由器，然后等1 至 2 秒后释放重置按钮，路由器闪烁呼吸式黄色 LED灯， 这时访问路由器pb-boot后台IP:192.168.1.1 你将看到一个 PandoraBox 固件升级页面，恭喜你成功输入pb-boot，俗称不死系统！

4，进入pb-boot后台手动刷入固件即可，目前可以刷入的系统：

  • 可以刷 padavan,也可以刷 pandorabox,以及 x-wrt系统。

5，很老的路由器CPU 性能太弱了，而且还是百兆网口，不建议做主路由使用。 刷机后可以添加机场，给矿机挖矿使用，也可以作为旁路由使用。 想折腾的网友可以试试刷入其他系统，自行测试！

三、刷回官方系统 ​

  • 1.下载官方固件，命名为miwifi.bin,放在u盘根目录
  • 2.路由器断电，插上u盘
  • 3.按住reset，可以使用牙签等，注意是按住，不是按一下，同时路由器插电，等待黄灯慢闪的时候松开reset，刷机完成会自动重启。
