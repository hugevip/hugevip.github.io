解决矿池连接问题，自建中转教程 ​

自建中转简单高效，防止二次抽水，目前各大矿池国内DNS基本被污染无法请求。

可搭建香港服务器转发数据解决该问题。

然后SSH连接云服务器，进入命令行界面。

1、安装转发软件 ​

复制命令粘贴至命令行回车执行

sudo apt-get update -y sudo apt-get install redir -y

2、一键启动转发服务 ​

sudo redir :转发端口 域名:端口

比如鱼池转发：

sudo redir :6687 eth.f2pool.com:6688

执行后没有任何输出，表示执行成功。

3、命令说明： ​

sudo redir :6687---监听访问本服务器的6687端口

eth.f2pool.com:6688---转发到eth.f2pool.com的6688端口

到此就搭建完成了！

连接服务器公网IP:6687即可挖矿。

例如：1.2.3.4:6687

停止转发服务：

查看当前监听端口的id

sudo netstat -anp |grep 6687

执行后，看到该监听端口id为1907（每台电脑都不一样）

停止当前转发服务

sudo kill 1907

此外，若云服务器重启，转发服务失效，需要重新按上面步骤操作启动，转发软件有时也会自动关闭，同上。
